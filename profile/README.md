# GovOn

> 동아대학교 현장미러형 연계 프로젝트 기반의 GovOn Agentic Shell.

## 소개

**GovOn**은 공공 업무 환경을 위한 terminal-first agentic shell입니다.

이 프로젝트는 **동아대학교 현장미러형 연계 프로젝트**로 진행되며, 실제 **현장기업 또는 공공기관과 연계한 문제 정의와 실행 환경**을 바탕으로 제품을 발전시키고 있습니다.

현재 제품 기준은 **GovOn Agentic Shell**이며, 첫 릴리즈(R1)는 다음 네 가지에 집중합니다.

- Interactive terminal shell
- Agent runtime
- Graph-based decision and orchestration
- Installable and offline package

웹/앱 UI 고도화는 R1 이후 범위로 두고, 실행 가능한 셸 경험과 배포 자산 정합성을 우선합니다.

## 프로젝트 성격

- Dong-A University field-mirrored collaboration project
- Public-sector and industry-linked execution context
- Terminal-first workflow
- Reproducible runtime and deployment assets
- Offline-ready distribution for restricted environments

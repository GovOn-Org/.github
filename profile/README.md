# GovOn (거브온)

> **Government + On** — 정부 서비스가 가동 중이라는 역동적인 느낌을 담은 이름입니다.

## 이름의 의미

**GovOn**은 두 가지 의미를 동시에 담고 있습니다:

- **Gov(ernment) + On**: 정부 서비스가 "켜져 있다", "가동 중이다"라는 의미로, 공공 서비스가 항상 시민 곁에서 작동하고 있다는 역동적인 이미지를 전달합니다.
- **민원온(民願 ON)**: 민원 업무를 "켠다", "시작한다"는 의미와 온디바이스(On-Device) AI의 **"온"** 이미지가 함께 살아 있습니다. 공공기관에서 사용하기에 적합한 사무적이면서도 친근한 이름입니다.

## 프로젝트 소개

**GovOn**은 온디바이스 AI 기반 민원 답변 자동 생성 시스템입니다.

LG EXAONE 대규모 언어 모델을 QLoRA 파인튜닝하고 AWQ 양자화하여, 공공기관의 제한된 하드웨어 환경(폐쇄망, GPU 서버)에서도 고품질 민원 답변을 실시간으로 생성할 수 있도록 경량화된 AI 솔루션을 제공합니다.

### 핵심 기술

| 기술 | 설명 |
|------|------|
| **EXAONE-3.5-7.8B** | LG AI Research 한국어 특화 LLM |
| **QLoRA** | 4-bit 양자화 기반 효율적 파인튜닝 |
| **AWQ** | Activation-aware Weight Quantization (4-bit) |
| **vLLM** | 고성능 LLM 추론 서빙 엔진 |
| **FAISS + RAG** | 벡터 검색 기반 검색 증강 생성 |
| **Figma MCP** | 디자인-개발 통합 프론트엔드 파이프라인 |

### 협업

- **동서대학교 소프트웨어융합대학** — AI 모델 개발 및 백엔드 엔지니어링
- **동서대학교 디자인학부** — UI/UX 디자인 (융합 캡스톤)
- **산업체 연계** — 현장미러형 프로젝트 (공공 민원 처리 시스템)
